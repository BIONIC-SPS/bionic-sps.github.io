# BIONIC

A cutting-edge research project pushing the boundaries of biological and computational integration.

## Landing Page

The landing page for the BIONIC research project is available in `index.html`. 

### Viewing the Landing Page

#### Local Development
To view the landing page locally, you can use any of these methods:

1. **Using Python (recommended)**:
   ```bash
   python3 -m http.server 8000
   ```
   Then open http://localhost:8000 in your browser.

2. **Using Node.js**:
   ```bash
   npx serve
   ```

3. **Direct File**: Simply open `index.html` in your web browser.

#### Deployment
The landing page can be easily deployed to:
- **GitHub Pages**: Enable GitHub Pages in repository settings and select the main branch
- **Netlify**: Connect your repository and deploy with zero configuration
- **Vercel**: Import the repository and deploy instantly
- Any static web hosting service
